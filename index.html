What Is This?

The Mod Maker is a visual node editor that lets you create custom content for Damage Roll and export it as a .json file the game can load directly.

You build your mod by connecting modules to a central root node — cards, decks, color themes, and visual overrides. When you're done, hit export and drop the file into the game's Mods panel.

No knowledge of JSON or code required. What you see is what you get.

Tips

01

Start at the Root

The center node is your mod's identity. Fill in the name, version, and author first — the export uses these as the filename.

Required
02

One Module Per Type

Each mod can only have one Cards module, one Theme module, etc. But a single mod can include all types at once — mix and match freely.

Structure
03

Card IDs Must Be Unique

The game will reject mods where a card ID collides with a built-in card (IDs 0–50) or another loaded mod. Keep your IDs above 1000 to be safe.

Cards
04

Theme vs Visuals

Theme vars override CSS variables game-wide. Visuals target specific UI elements. Use theme for a full color overhaul, visuals for surgical changes.

Styling
05

Preview Before Export

Use the { } Preview button to inspect the raw JSON before downloading. It's a good sanity check — especially for deck card lists.

Workflow
06

Pan & Zoom Freely

Middle-drag or click the background to pan the canvas. Scroll to zoom. Use the ⊙ button in the bottom-right to re-center the view at any time.

What Mods Can Do

Custom Cards

New cards with any emoji icon, class, rarity, multiplier, and description. Unlimited.

Allowed
📦

Custom Decks

Build decks from built-in or modded cards. Up to 9 cards per deck. Unlimited decks.

Allowed
🎨

Color Themes

Override any of 34 CSS color variables. Appears as a swatch in the game's Settings panel.

Allowed
✨

Visual Overrides

Target specific UI elements — HP bars, buttons, cards, panels — with custom CSS values.

Allowed
🤖

AI & Mechanics

Opponent AI behaviour, difficulty, crit multipliers, and core game rules cannot be modded.

Blocked
⚙️

Game Settings

Audio, controls, and other player settings are off-limits to mods.

Blocked
